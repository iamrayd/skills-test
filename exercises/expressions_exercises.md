# Expressions Exercises (Infix ↔ Postfix)

**Infix → Postfix**
1) `A + B * C`
2) `(A + B) * (C + D)`
3) `A ^ B ^ C`  (note: `^` is right-associative)
4) `-A + (B - C) * D`
5) `3 + 4 * 2 / (1 - 5) ^ 2 ^ 3`

**Postfix → Infix**
1) `A B + C D + *`
2) `A B C * + D E / -`
3) `A B ~ * C +`
4) `x y z + ^ a b / -`

> Hint: For unary minus in postfix, we use `"~"`.
